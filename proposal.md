# Vibe Check — Proposal

## What are you building? Who is it for?
Vibe Check is a platform that will recommend movies and music based on the mood that you are in. The user will choose the mood/vibe that they are currently in (for example: cozy, happy, sad, hype, etc.) and they will get back a list of movies or playlist for the mood that they are in. It is designed for people that don't always have a playlist for every occasion or spend too much time scrolling through movie options without being able to find one that they want to watch. 

## Why?
I often find myself struggling to find the right music to listen to or the right movie to watch. It is a tool that I have always wished to have. Therefore, I believe that other people out there must have the same problem that I often do. It can be overwhelming, especially the movie aspect of it, and this will help make it easier. 

## MVP vs. Stretch Goals
**MVP:**
- Mood selector on the home page with an option to choose between movies or playlists
- Movie results pulled from the TMDB API
- Embedded Spotify playlist per vibe
- Saved watchlist using localStorage

**Stretch Goals:**
- Color scheme shifts to match the selected vibe
- Search within results
- User can rate or review saved movies

## Technologies
- HTML, CSS, JavaScript
- TMDB API (movie data)
- Spotify embedded playlists (no key required)
- localStorage for watchlist persistence
