# Vibe Check 

A mood-based movie and music recommender. Its easy, pick a vibe, choose between movies or music, and get personalized recommendations instantly

[Insert your GitHub Pages link here]

## Features
- 6 mood/vibe options to choose from
- Movie recommendations pulled from the TMDB API
- Embedded Spotify playlists matched to each vibe
- Save movies to a personal watchlist
- Remove movies from your watchlist
- Light and dark mode toggle
- Vibe-based color themes
- Fully responsive on mobile and desktop

## Technologies Used
- HTML, CSS, JavaScript
- TMDB API for movie data
- Spotify Embed for music playlists
- localStorage for watchlist and theme persistence
- Google Fonts (DM Sans)
- GitHub Pages for deployment
